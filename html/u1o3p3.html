Unit 1 - Outcome 3.3

Develop solutions to improve own productivity using IT in digital projects.

If I had the chance for a do-over, I’m not sure that there’s much I would do differently. The whole project has gone smoothly, and the end result is one that I’m proud of.

The main change I’d make is simply to make better use of time so that I can put a little extra work into every aspect of the project. After a certain point there are diminishing returns, but I still could have benefitted from spending a little extra time with the computer, polishing whatever possible.

To get the extra time, I could easily use Github to work on the project from home, since HTML, CSS and other components of the site are fully accessible via my computer at home through Github. In fact, most of the programs that I’ve used are available both for Mac and Windows (which my PC uses), so I could use the same software as if I’d never left the Mac in the first place.
