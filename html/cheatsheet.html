Cheat sheet

HTML

  • <!DOCTYPE html>
  • <html lang="en">
  • <head meta charset="UTF-8">
  • <head name="viewport" content="width=device-width, initial-scale=1.0">
  • <meta http-equiv="X-UA-Compatible" content="ie=edge">
  • <link rel="stylesheet" href="../folder/cssdocname.css" />
  • <title>Title that appears in the tab name</title>
  • <body>Everything between the head and the HTML end</body>
  • <h1> to <h9> for headings, <p> for paragraphs
  • <q> for quotes, <code> for code
  • <ul> for unordered lists, <ol> for ordered lists, <li> for list items
  • <a> for links and href="x" for the url
  • <pre> for preserving line breaks and indents (ascii art)
  • <script> for internal javascript or external javascript linking
  • (type="text/javascript" src="../folder/jsfilename.js")
  • <sub> and <sup> for subscript and superscript
  • <quote> for inline, <blockquote> for blocks of quoted text
  • <div> divides the page, necessary for decent layout
  • <em> for emphasis, or <i> for italics
  • <frame>s are deprecated, try not to use them
  • <canvas> is a drawable area for use with javascript
  • <del> and <ins> for deleted and insterted text (file history)
  • for inline style, (don't) use style="css content" in a tag
  • use id="id" to add an id to a tag. Conventionally, use 1 of each id
  • use class="class" for anything that has more than 1 instance
  • class becomes className when being accessed via javascript

CSS

  • htmlelement { key: value }
  • .class { key: value }
  • #id { key: value }
  • * { applied to all elements }
  • color: color of text
  • background-color: background color
  • colors accept rgb, rgba, #fff, #ffffff, #fffa, #ffffffaa, hsv, hsl, hwb, cmyk, color names
  • inner to outer: size, padding, border, margin
  • sizes accepted: number, percent, cm, mm, in, pt, px, vw, vh (viewport width, height)
  • background-image: linear-gradient(to top right, #fff0, #ffff) (trans-white to white, to top right)
  • direction for linear-gradient can include degrees (15deg)
  • background-image: linear-gradient(#fff0, #ffff 10%)
  • the 10% means that that colour will start at 10% of the gradient area. This is 90% solid (roughly)
  • if an absolute number is used instead (px) then it'll start at that length from the start of the previous color
  • use border-radius for corners, 25px will generally be noticably rounder
  • border: width style color (style can be none, hidden, solid, double, dotted, dashed...)
  • opacity: opacity of the element, including text
  • font: style variant weight stretch size/line-height font-family
  • font-style: normal, italic, oblique (xdeg)
  • variant: used for ligatures and caps state (like small caps)
  • font-weight: how bold text is, from lighter to bolder (or numbers)
  • line-height: how spaced out the lines are from each other
  • font-family: once defined, changes the font
  • float: 'float: left' will move the object to the left of its parent inline with text
