Unit 1 - Outcome 2.2

Describe whether the IT tools selected were appropriate.

In case you couldn’t tell from the tone of the last page, the tools I’ve used were definitely appropriate to their task. Everything I’ve used during the course of this project has performed beyond my expectations.

The only reason I can imagine to call any of these tools ‘inappropriate’ is that Trello is somewhat lacking in security. Well, it has security, but there are also those that will breach it and steal information. Thankfully, I didn’t succumb to any kind of security breaches as far as I know, but if security were a serious concern in another project then I’d probably pick an alternative.
